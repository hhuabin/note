# 分支

1. 查看状态：**切换分支前，一定要查看状态，确认所有内容已提交再切换**

     ```bash
     git status
     ```

2. 新建分支

     ```bash
     git branch branchName
     ```

3. 查看分支

     ```bash
     git branch -v
     ```

4. 切换分支

     ```bash
     git checkout branchName
     ```

5. 新建分支并且直接切换到新分支

     ```bash
     git checkout -b branchName
     ```

6. 合并分支

     ```bash
     git merge branchName
     ```

7. 修改分支名字

     ```bash
     git branch -M newBranchName
     ```

8. 删除分支

     ```bash
     git branch -d branchName
     ```

9. 强行删除分支

     ```bash
     git branch -D branchName
     ```



# `push`代码步骤

1.  **`git pull`**（拉取远端最新代码）
2.  **`git commit`**（提交你的本地改动）
3.  **`git push`**（推送到远端）

# 远程分支

1. 查看所有远程分支

   ```bash
   git branch -r
   ```

2. 查看本地和远程的所有分支

   ```bash
   git branch -a
   ```

3. 如果已经有远程分支，本地没有的话

    1.   不用创建，直接切换分支了。Git 会自动检测远程有 `origin/dev`，然后提示创建跟踪分支

         ```bash
         git checkout dev
         
         # 或者
         git switch dev
         ```

    2.   或者：创建本地 `dev` 分支并自动关联到远程的 `origin/dev`

         ```bash
         git checkout -b dev origin/dev
         ```

4. 不创建本地分支，直接 `merge` 远程分支

    ```bash
    git pull origin dev
    ```

# `::`

```python
array[start:stop:step]
```

默认值：

- `start = 0`
- `stop = len(a)`
- `step = 1`，**step不能为0**

| 写法                 | 含义                  |
| -------------------- | --------------------- |
| `a[:]`               | 整个切片              |
| `a[::-1]`            | 倒序                  |
| `a[::2]`             | 隔一个取一个          |
| `a[1::2]`            | 从 1 开始隔一个取一个 |
| `a[::]`              | 整个 + 默认步长       |
| `a[start:stop:step]` | 标准切片语法          |



## 倒序（最常见）

```python
a = [1, 2, 3, 4]
print(a[::-1])  # [4, 3, 2, 1]
```



## 正序

```python
a = [0, 1, 2, 3, 4, 5]
print(a[::2])  # [0, 2, 4]
```

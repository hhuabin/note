# 常见异常类型

- `Exception`：通用异常
- `ValueError`：值错误
- `TypeError`：类型错误
- `IndexError`：索引错误
- `KeyError`：键错误
- `FileNotFoundError`：文件未找到
- `ZeroDivisionError`：除零错误
- `AttributeError`：属性错误
- `ImportError`：导入错误



# `raise` 抛出异常

```python
def validate_age(age):
    if age < 0:
        raise ValueError("年龄不能为负数")
    if age > 150:
        raise CustomError("年龄不合理")
    return True
```



# `try` 捕获异常

```python
try:
    # 尝试执行的代码块
except ExceptionType1:
    pass   #　跳过异常
    # 捕获特定类型异常
except ExceptionType2 as e:
    # 捕获并命名异常对象
else:
    # 如果try中没有发生异常，执行这里
finally:
    # 无论是否出错，都会执行
```

```python
try:
    result = 10 / 2
except ZeroDivisionError as e:
    print("除零错误：", e)
else:
    # 可以继承 try 的 result
    print("没有出错，结果是：", result)
finally:
    print("无论如何都会执行")
```

### 捕获多个异常

```python
try:
    a = int("abc")
except (ValueError, TypeError):
    print("捕获 ValueError 或 TypeError")
```

### 捕获异常对象：可以使用 `as` 获取异常对象，查看错误信息

```python
try:
    1 / 0
except ZeroDivisionError as e:
    print("异常信息：", e)
```

### 捕获所有异常

```python
try:
    x = 1 / 0
except Exception as e:
    print("捕获所有异常：", e)
```





## 嵌套 try

```python
try:
    try:
        1 / 0
    except ZeroDivisionError:
        print("内层异常")
        raise  # 重新抛出异常
except Exception:
    print("外层捕获")
```

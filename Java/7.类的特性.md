# 类 修饰符   

| 修饰符                          | 说明                       |
| ------------------------------- | -------------------------- |
| `public                       ` | 公共类，可以被任何对象访问 |
| `abstract`                      | 抽象类，只能被继承         |
| `final`                         | 最终类，**不能被继承**     |
| `缺省`                          | 在相同包下才能访问         |



# 方法的重载

Java 可以定义多个同名的方法，只要这些方法的参数不同即可。



# 构造方法

- 构造方法名与类名完全相同

- Java编译器会为自动为类生成默认的构造方法，创建对象时，会自动调用默认构造方法

  如果类中有有参数的构造方法，则必须显示为类添加默认构造方法

- 构造返回值是该**类本身**，故不能用void修饰构造方法

- 构造方法也可以重载

- 在构造方法中调用另一构造方法时，this() 必须放在第一行

- 继承时，如果父类中只有带参数的构造方法，则子类中必须有 super(...args)



# `static`静态成员

被 `static` 修饰的成员成为静态成员

让成员“属于类”，而不是属于对象



## 1.`static`静态变量

```java
class User {
    static int count = 0;
}
```

访问方式：

- 类名.静态变量名（建议使用这个）

  ```java
  User.count++;
  ```

- 对象名.静态变量名

  ```java
  User user = new User();
  System.out.println(user.count);   // 0
  ```



### `static` 继承

```java
class Parent {
    static int A = 10;
}

class Child extends Parent {}

System.out.println(Child.A); // 10
```



### 继承中的 `static` 字段隐藏（方法隐藏类似）

子类可以把父类的`static`字段隐藏掉，只有访问父类才可以访问到

```java
class Parent {
    static int A = 10;
}

class Child extends Parent {
    static int A = 20;
}

System.out.println(Parent.A); // 10
System.out.println(Child.A);  // 20
```

对比没有`static`的情况

```java
class Parent {
    int A = 10;
}

class Child extends Parent {
    int A = 20;
}

Parent p = new Child();
System.out.println(p.A);   // 20
```



## 2.静态方法

静态方法只能访问**静态变量**或者**静态方法**

```java
class Utils {
    static int add(int a, int b) {
        return a + b;
    }
}
```

访问： 
- 类名.静态方法名

  ```java
  Utils.add(1, 2);
  ```

- 对象名.静态方法名

  ```java
  Utils utils = new Utils();
  utils.add(1, 2);
  ```



## 静态初始化器

- 只要创建类时执行，创建实例类不执行，即与创建多少个实例化对象无关

```java
static {
    PI = 3.14159265
}
```



# `final`最终成员

👉 **让东西“不能被改变 / 不能被重写 / 不能被继承（final类不可以被继承，变量和方法都可以被继承）”**

## 1.`final`变量

`final` 只能赋值一次，本质是常量，限制的是**“引用不变”**

```java
final int a = 10;

final List<String> list = new ArrayList<>();
list.add("a");              // ✅
list = new ArrayList<>();   // ❌
```



### `final`变量的继承

```java
class Parent {
    final int A = 10;
}

class Child extends Parent {}

Child c = new Child();
System.out.println(c.A); // 10
```



### `final`变量继承中的隐藏（注意不是重写）

两个类中的 `A` 是两块独立内存

```java
class Parent {
    final int A = 10;
}

class Child extends Parent {
    final int A = 20;
}


Child c = new Child();
System.out.println(((Parent)c).A);  // 10
System.out.println(c.A);            // 20
```



## 2.`final`方法

### `final`方法的继承

```java
class Parent {
    final void test() {
        System.out.println("Parent");
    }
}

class Child extends Parent {}

Child c = new Child();
c.test();          // ✅
```



### `final`方法继承中不可以被重写/隐藏

```java
class Parent {
    final void test() {
        System.out.println("Parent");
    }
}

// 子类不能重写
class Child extends Parent {
    final void test() {              // ❌
        System.out.println("Parent");
    }
}
```

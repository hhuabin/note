# 类的继承



## `java.lang.Object`

`java` 中有一个名为 `java.lang.Object` 的特殊类，所有的类都是直接或间接继承该类得到的

`java.lang.Object` 的常用方法

```java
// 判断两对象是否为同一对象（根据地址）
public boolean equals(Object obj) { return (this == obj); }

// 将对象转为字符串
public String toString();

// 返回运行时对象所属的类
public final native Class<?> getClass();

// 返回对象的一个副本
protected native Object clone();

// 获取对象的 hashCode
public native int hashCode();
```



对象运算符 **`instanceof`**

使用： 引用类型变量 `instanceof` 引用类型

```java
Object str = "aaa";
if(str instanceof String){};
```



## 类成员的继承

通过 `extends` 关键字可以将父类中的**非私有成员**继承给子类

- Java在执行子类的构造方法之前，会自动调用父类的无参构造方法
- 调用父类构造方法的 `super()` 语句必须写在子类构造方法的第一行

| 修饰符            | 是否继承 | 子类是否可访问  | 是否可重写 | 特殊说明                       |
| ----------------- | -------- | --------------- | ---------- | ------------------------------ |
| `public`          | ✅        | ✅               | ✅          | 对外 API                       |
| `protected`       | ✅        | ✅（跨包需继承） | ✅          | 给子类扩展                     |
| `default`（包内） | ✅        | ✅（同包）       | ✅（同包）  | 包内协作                       |
| `private`         | ❌        | ❌               | ❌          | 内部实现                       |
| `static`          | ✅        | ✅               | ❌          | ✅可隐藏                        |
| `final`           | ✅        | ✅               | ❌          | ✅变量可隐藏<br />❌方法不可隐藏 |

### 继承中 `private`

```java
class Parent {
    private int a = 10;
}

class Child extends Parent {}

Child c = new Child();
System.out.println(c.a);   // ❌ 不可以访问父类的 private
```

子类可以定义与父类`private`同名的变量

```java
class Child extends Parent {
    int a = 20;
}
System.out.println(c.a);  // 20
```

# 遍历方式

## 前序遍历

**顺序**：根节点 -> 左子树 -> 右子树

```
    1
   / \
  2   3
 / \
4   5
```

- 前序遍历结果：`1, 2, 4, 5, 3`



## 中序遍历

**顺序**：左子树 -> 根节点 -> 右子树

```
    1
   / \
  2   3
 / \
4   5
```

- 中序遍历结果：`4, 2, 5, 1, 3`



## 后续遍历

**顺序**：左子树 -> 右子树 -> 根节点

```
    1
   / \
  2   3
 / \
4   5
```

- 后序遍历结果：`4, 5, 2, 3, 1`

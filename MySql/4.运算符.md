# 比较运算符

| 运算符 | 名称     | 示例                              |
| ------ | -------- | --------------------------------- |
| =      | 等于     | SELECT C FROM TABLE WHERE A = B   |
| <=>    | 安全等于 | SELECT C FROM TABLE WHERE A <=> B |
| <>(!=) | 不等于   | SELECT C FROM TABLE WHERE A <> B  |
| <      | 小于     | SELECT C FROM TABLE WHERE A < B   |
| <=     | 小于等于 | SELECT C FROM TABLE WHERE A <= B  |
| >      | 大于     | SELECT C FROM TABLE WHERE A > B   |
| >=     | 大于等于 | SELECT C FROM TABLE WHERE A >= B  |

| 运算符      | 名称             | 示例                                        | 作用 |
| ----------- | ---------------- | ------------------------------------------- | ---- |
| IS NULL     | 为空运算符       | SELECT B FROM TABLE WHERE A IS NULL         |      |
| IS NOTMULL  | 不为空运算符     | SELECT B FROM TABLE WHERE A IS NOTMULL      |      |
| LEAST       | 最小值运算符     | SELECT D FROM TABLE WHERE C LEAST(A, B)     |      |
| GREATEST    | 最大值运算符     | SELECT D FROM TABLE WHERE C GREATEST(A, B)  |      |
| BETWEEN AND | 两值之间的运算符 | SELECT D FROM TABLE WHERE C BETWEEN A AND B |      |
| ISNULL      | 为空运算符       | SELECT B FROM TABLE WHERE A ISNULL          |      |
| IN          | 属于运算符       | SELECT D FROM TABLE WHERE C IN(A, B)        |      |
| NOT IN      | 不属于运算符     | SELECT D FROM TABLE WHERE C NOT IN(A, B)    |      |
| LIKE        | 模糊匹配运算符   | SELECT C FROM TABLE WHERE A LIKE B          |      |
| REGEXP      | 正则表达式运算符 | SELECT C FROM TABLE WHERE A REGEXP B        |      |

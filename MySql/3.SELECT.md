# SELECT

```sql
# 查询表的所有数据
SELECT * FROM user;

# DUAL 伪表
SELECT 1 + 1 FROM DUAL;
```



## `SELECT`语句中的关键字

### (字段)列别名

查出来的列表头会显示列别名

**列别名只能在`ORDER BY`中使用，不能在`WHERE`中使用**

1.   跟在列明后面

     ```sql
     SELECT name n FROM user;
     ```

2.   `AS`(alias)关键字

     ```sql
     SELECT name AS n FROM user;
     ```

3.   `""`双引号，不要使用单引号（比第一种可以支持更多场景，比如可以支持空格了！）

     ```sql
     SELECT name "n a m e" FROM user;
     ```



### `DISTINCT`去重

```sql
SELECT DISTINCT name FROM user;
```

不要进行多字段去重

```sql
# 报错
SELECT name, DISTINCT age FROM user;

# 查出来也没有意义
SELECT DISTINCT name, age FROM user;
```



### `null`空值参与运算

```sql
# 如果 age 是NULL 则使用0
IFNULL(age, 0)
```



### 着重号` `` `

```sql
# 兼容名字
SELECT * FROM `user-user`;

# 标注非关键字
SELECT * FROM `order`;
```



## `WHERE`过滤

```sql
SELECT * FROM user WHERE age = 18;
```



## `ORDER BY`排序

### 单列排序

```sql
# 默认升序
SELECT * FROM t_emp
ORDER BY emp_id;

# ASC 升序
SELECT * FROM t_emp
ORDER BY emp_id ASC;

# DESC 降序
SELECT * FROM t_emp
ORDER BY emp_id DESC;
```

### 多列排序

```sql
# , 分隔
# 先排 emp_id，emp_id有重复的再根据 emp_salary DESC 排列
SELECT * FROM t_emp
ORDER BY emp_id, emp_salary DESC;
```



## `LIMIT`分页

```sql
LIMIT [位置偏移量,] 行数
```

```sql
SELECT * FROM table
LIMIT (PageNo - 1)*PageSize, PageSize;
```

```sql
# 从第11条开始查10条（0是第一条）
SELECT * FROM 表名 LIMIT 10, 10;
```

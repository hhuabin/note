# SELECT 查询

通用查询模板

```sql
SELECT column1, column2, ...       # 查询字段 SELECT name, age
FROM tableName                     # 表名 FROM user
[(LEFT / RIGHT) JOIN table2 ON condition]      # 连表 JOIN employee ON user.id = employee.id
[WHERE condition]                  # 连表条件 WHERE salary > 5000
[GROUP BY column]                  # 分组 GROUP BY user
[HAVING condition(聚合函数)]        # 分组后过滤 HAVING COUNT(*) > 3
[ORDER BY column ASC|DESC, ...]    # 排序 ORDER BY age DESC, name ASC;
[LIMIT offset, size];              # 分页 LIMIT 0, 10;
```

##### 常见聚合函数

```sql
COUNT()
SUM()
AVG()
MAX()
MIN()
```

##### `offset = (pageNum - 1) * pageSize`

| 变量       | 说明             |
| ---------- | ---------------- |
| `offset`   | 起始索引(偏移量) |
| `PageSize` | 每页条数         |



## 多表查询

1. 内连接（交集）

   ```sql
   SELECT ...
   FROM A
   INNER JOIN B ON A.id = B.a_id;
   ```

2. 左外连接（左表全保留）

   ```sql
   SELECT ...
   FROM A
   LEFT JOIN B ON A.id = B.a_id;
   ```

3. 右外连接

   ```sql
   SELECT ...
   FROM A
   RIGHT JOIN B ON A.id = B.a_id;
   ```

4. 满外连接

   ```sql
   # Oracle用
   SELECT ...
   FROM A
   FULL OUTER JOIN B ON A.id = B.a_id;
   ```

   ```sql
   SELECT ...
   FROM A
   LEFT JOIN B ON A.id = B.a_id
   
   UNION     # 去重
   # UNION ALL      # 不去重
   
   SELECT ...
   FROM A
   RIGHT JOIN B ON A.id = B.a_id;
   ```



## `SELECT`语句中的关键字

1. `DISTINCT`（去重）

   ```sql
   SELECT DISTINCT name FROM user;
   ```

2. `AS`（别名）

   ```sql
   # 列别名
   SELECT name AS username FROM user;
   ```

   ```sql
   # 表别名
   SELECT u.name FROM user u;
   ```

3. `AND / OR / NOT`（条件组合）

   ```sql
   WHERE age > 18 AND status = 1
   ```

4. `IN / NOT IN`

   ```sql
   WHERE id IN (1, 2, 3)
   ```

5. `BETWEEN`

   ```sql
   WHERE age BETWEEN 18 AND 30
   ```

6. `LIKE`（模糊查询）

   ```sql
   WHERE name LIKE '%黄%'
   ```

7. `IS NULL / IS NOT NULL`

   ```sql
   WHERE email IS NULL
   ```

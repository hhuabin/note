# 视图

**视图的增、删、改操作与表一致**

1. 创建视图

   ```sql
   CREATE VIEW viewName
   AS (子查询)
   ```

2. 查看数据库表、视图对象

   ```sql
   SHOW TABLES;
   ```

3. 查看视图

   ```sql
   DESCRIBE viewName;
   ```

4. 查看视图属性

   ```sql
   SHOW TABLES STATUS LIKE viewName\G
   ```

5. 查看视图详细定义信息

   ```sql
   SHOW CREATE VIEW viewName;
   ```

6. 删除视图

   ```sql
   DROP VIEW IF EXISTS viewName;
   ```

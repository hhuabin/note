# 表管理

1. 创建表（注意数据库/表的字符集）

   ```sql
   CREATE TABLE IF NOT EXISTS tableName (
       id INT PRIMARY KEY,
       user_name VARCHAR(50) NOT NULL,
       age INT,
       email VARCHAR(100),
       create_time DATETIME
   );
   ```

   当主键是多个字段，可以把主键写在最后

   ```sql
   CREATE TABLE IF NOT EXISTS user_role (
       user_id BIGINT,
       role_id BIGINT,
       create_time DATETIME,
       PRIMARY KEY (user_id, role_id)
   );
   ```

2. 显示表结构

   列出表字段的相关信息，如字段的类型

   ```sql
   DESCRIBE tableName;
   
   DESC tableName;
   ```

3. 修改表的字符集

   ```sql
   alter table tableName charset utf8
   ```

4. 删除表

   ```sql
   DROP TABLE tableName;
   ```

5. 重命名表

   ```sql
   ALTER TABLE tableName RENAME TO newTableName;
   ```

6. 清空表数据

   ```sql
   TRUNCATE TABLE tableName;      # 快，重置自增 ID，但是不可回滚
   ```

   ```sql
   DELETE FROM tableName;         # 逐行删除，可回滚
   ```

7. 修改列限制

   ```sql
   ALTER TABLE tableName MODIFY columnName VARCHAR(100) NOT NULL;
   ```

8. 重命名该列

   ```sql
   ALTER TABLE tableName RENAME COLUMN oldName TO newName;
   ```

   ```sql
   ALTER TABLE tableName CHANGE oldName newName VARCHAR(100);     # MySQL（老版本）写法
   ```

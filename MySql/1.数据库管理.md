# 数据库操作

1. 数据库版本

   ```sql
   mysql --version
   ```

2. 连接数据库：mysql -h 主机名 -P 3306 -u 用户名 -p 用户密码

   ```sql
   mysql -h localhost -P 3306 -u root -p
   ```

3. 退出登录

   ```sql
   quit;
   
   ## or
   exit;
   ```

4. 启动服务

   ```sql
   net start mysql80
   ```

5. 停止服务

   ```sql
   net stop mysql80
   ```

6. 显示所有数据库

   ```sql
   SHOW DATABASES; 
   ```

7. 导入数据库

   ```sql
   # 命令行执行
   source d:\data.sql;        # source sql文件的硬盘绝对路径;
   
   ```

8. 查看数据库编码（需要utf8之类的才可以支持中文，创建表时会默认使用该字符集）

   ```sql
   show variables like 'character_%';
   
   SHOW VARIABLES like 'collation_%'
   ```
   
   

# 数据库管理

1. 创建数据库

   ```sql
   CREATE DATABASE databaseName;
   
   CREATE DATABASE IF NOT EXISTS databaseName CHARACTER SET 'utf8';
   ```

2. 查看数据库（会列出该连接下的数据库名）

   ```sql
   SHOW DATABASES;
   ```

3. 查看数据库信息（可以查看数据库的编码信息等）

   ```sql
   SHOW CREATE DATABASE databaseName;
   ```

4. 修改数据库字符集

   ```sql
   # 查看数据库编码
   show variables like 'character_%';
   
   SHOW VARIABLES like 'collation_%'
   ```

   ```sql
   ALTER DATABASE databaseName CHARACTER SET utf8mb4;
   ```

5. 删除数据库

   ```sql
   DROP DATABASE databaseName;
   ```

6. 查看当前处于哪个数据库中

   ```sql
   SELECT DATABASE();
   ```

7. 使用该数据库

   ```sql
   USE databaseName;
   ```




# SQL 分类

1.   DDL：数据定义语言。`CREATE` 、`ALTER`、`DROP`、`RENAME`、`TRUNCATE`
2.   DML：数据操作语言。`INSERT`、`DELETE`、`UPDATE`、`SELECT`
3.   DCL：数据控制语言。`COMMIT`、`ROLLBACK`、`SAVEPOINT`、`GRANT`、`REVOKE`



# SQL规则

1.   3种注释规则

     ```sql
     # 单行注释
     -- 单行注释（--后必须有空格）
     
     /*
      *  多行注释
      */
     ```

2.   数据库、表名不得超过30个字符，变量名限制为29个

3.   必须只能包含A-Z,a-z,0-9，共63个字符

4.   数据库名、表名、字段名等对象名中间不要包含空格



# SQL规范

1.   字符串型和日期时间类型的数据可以使用单引号(`''`)表示
2.   列的别名，尽量使用双引号(`""`)，而且不建议省略
3.   数据库名、表名、字段名使用**下划线风格**，不使用驼峰；windows下不区分大小写会带来实际问题
4.   某些傻逼会使用`-`中划线连接，如`user-name`，这时候就需要使用**着重号` `` `**；只要用了 `-`，以后几乎所有 SQL 都得加引号

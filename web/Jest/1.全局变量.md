## `describe(name, fn)`

`describe(name, fn)` 创建一个块，将几个相关的测试分组在一起

```typescript
describe('arrayUtils', () => {
    test('isArray returns true for arrays', () => {
        expect(isArray([])).toBe(true)
    })
})
```



# 钩子函数

`beforeAll`：整个测试文件（或 describe）开始前，只执行一次

`beforeEach`：每个 `test` 执行前都会执行

`afterEach`： 每个 `test` 执行后都会执行

`afterAll`：整个测试文件（或 describe）结束后，只执行一次

作用范围：四个钩子函数都是当前文件（写在`describe`外） / 当前 `describe` 中生效，不污染其他文件

1. `beforeAll` 定义在文件级，当前文件级只执行一次，如果 `describe` 中还有定义 `beforeAll`，也会被执行
2. `beforeEach` 定义在文件级，当前文件的所有方法优先执行全局 `beforeEach` 再执行 `describe` 里面的 `beforeEach`
3. 总的来说都是先执行外部的，再执行内部的函数

```typescript
beforeAll(() => {
    console.log('beforeAll')
})

beforeEach(() => {
    console.log('beforeEach')
})

afterEach(() => {
    console.log('afterEach')
})

afterAll(() => {
    console.log('afterAll')
})
```

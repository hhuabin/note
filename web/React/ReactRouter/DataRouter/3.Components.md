## `ScrollRestoration`

[ScrollRestoration](https://reactrouter.com/6.28.0/components/scroll-restoration) 滚动恢复

```tsx
const RootRouteLayout: React.FC = () => {
    return (
        <>
            <ScrollRestoration
                getKey={(location, matches) => {
                    return location.key                            // ScrollRestoration 默认返回值；每次导航都是独立 scroll，只有路由返回才会记录
                    // return Date.now().toString()                      // 每次 key 都不同，scroll滚动条会被重置到顶部
                    // return location.pathname + location.search        // 同个 url 下，滚动条位置不变
                }}
            />
            <Outlet />
        </>
    )
}
```

# :star: `Hooks`

## 1.`useNavigate` 编程式导航（跳转路由）

[useNavigate](https://reactrouter.com/6.28.0/hooks/use-navigate)

```typescript
const navigate = useNavigate()

navigate("/home")          // 跳转
navigate(-1)               // 返回上一页
navigate("/login", { replace: true }) // 替换历史
```



## 2.`useSearchParams` 操作 URL query 参数

```typescript
const [searchParams, setSearchParams] = useSearchParams()

const page = searchParams.get("page")

setSearchParams({ page: "2" })
```

## 5.`useMatches` 访问路由数据

获取当前命中的整个路由链；[useMatches](https://reactrouter.com/6.28.0/hooks/use-matches "useMatches") 用法

1. 获取父路由数据（非常重要）

   比 `context` 更方便

   ```typescript
   const matches = useMatches()
   
   const rootData = matches.find(m => m.id === "root")?.data
   ```

2. 做面包屑（`breadcrumb`）

   ```typescript
   const crumbs = matches.map(m => m.handle?.crumb)
   ```

3. 控制页面行为（比如 `scroll`）

   ```typescript
   <>
       <ScrollRestoration
           getKey={(location, matches) => {
               return matches[matches.length - 1].id
           }}
       />
       <Outlet />
   </>
   ```

4. 做权限 / 布局控制

   ```typescript
   const matches = useMatches()
   
   const needAuth = matches.some(m => m.handle?.auth)
   ```

```typescript
// useMatches 的返回值，里面有 handle
[
    {
        id: string
        pathname: string
        params: object
        data: any
        handle: any
    }
]
```

## 6.`useNavigation`

[useNavigation](https://reactrouter.com/6.28.0/hooks/use-navigation) 获取当前“导航状态”（加载中？提交中？）

常见用法

1. 全局 `loading`；不建议做

   ```typescript
   const GlobalSkeletonLayout: React.FC = () => {
   
       const navigation = useNavigation()
   
       if (navigation.state === 'loading') {
           return <Skeleton />
       } else {
           return <Outlet />
       }
   }
   ```

2. 按钮 `loading`

   ```tsx
   <button disabled={navigation.state === "submitting"}>
     提交中...
   </button>
   ```

# 当前页跳转

```javascript
window.location.href = 'url'         // 增加历史记录

const a = document.createElement('a')
a.href = '/'
a.click()
```

```typescript
window.location.replace('url')       // 替换当前历史记录，函数
```



# 新页面跳转

```javascript
window.open('url', '_blank')
```

```typescript
const a = document.createElement('a')
a.href = '/'
a.target = '_blank'
a.click()
```



# 返回跳转至首页

```typescript
// 该方法需要保证当前页面是路由栈最后一页，不然就只能用别的方式
navigate(-(window.history.length - 1))
```

```typescript
navigate(-3)
```

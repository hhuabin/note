# 内存泄漏

内存泄漏（Memory Leak）本质是：**程序中已经不再需要的内存，没有被释放，仍然被“引用”着**，导致这部分内存一直占用，时间久了就会越积越多



## 内存泄漏原因及改正

### 1.全局变量 / 意外挂载到 window

```javascript
function fn() {
    data = new Array(1000000) // 没有声明
}
```

解决办法：避免隐式全局变量

```javascript
function fn() {
    const data = []
}
```



### 2.闭包导致的大对象无法释放

```javascript
function createHandler() {
      const bigData = new Array(1000000)

      return function () {
          console.log(bigData.length)
      }
}
const handler = createHandler()
```

解决办法：手动释放

```javascript
let handler = createHandler()
handler = null
```



### 3.定时器未清理

```javascript
useEffect(() => {
    const timer = setInterval(() => {}, 1000)
}, [])
```

解决办法：清理定时器

```javascript
useEffect(() => {
    const timer = setInterval(() => {}, 1000)
    return () => clearInterval(timer)
}, [])
```



### 4.事件监听未移除

```javascript
window.addEventListener('resize', handler)
```

解决办法：移除监听

```javascript
useEffect(() => {
    const handler = () => {}

    window.addEventListener('resize', handler)
    return () => window.removeEventListener('resize', handler)
}, [])
```



### 5.DOM 引用未释放

```javascript
const el = document.getElementById('app')
```

如果 DOM 被删除，但 JS 还持有 `el`，DOM 不会被 GC

解决办法：删除引用

```javascript
let el = document.getElementById('app')
el = null
```



### 6.缓存（Map / Array）无限增长

```javascript
const cache = new Map()

function add(key, value) {
    cache.set(key, value)
}
```

一直加，不删除，强引用一直存在

解决办法：限制大小（LRU）

```javascript
if (cache.size > 1000) {
    cache.clear()
}
```



### 7.未取消的异步请求

```javascript
useEffect(() => {
    fetch('/api').then(res => setData(res))
}, [])
```

解决办法：`AbortController`

```javascript
useEffect(() => {
    const controller = new AbortController()

    fetch('/api', { signal: controller.signal })

    return () => controller.abort()
}, [])
```



### 8.React 中的“隐性泄漏”

```javascript
const [data, setData] = useState(hugeObject)

const ref = useRef(hugeData)
```

解决办法，用完置空

```javascript
useEffect(() => {
    return () => {
        setData(null)
        ref.current = null
    }
}, [])
```



### 9.第三方库 / 未销毁实例

```javascript
const chart = new Chart(ctx)
```

解决办法：记得销毁

```javascript
chart.destroy()
```

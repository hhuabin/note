# 微信小程序不用做事件委托

如果硬要做，可以使用`mark`

```wxml
<view bindtap="handleClick">
    <view
        wx:for="{{list}}"
        wx:key="id"
        class="item"
        data-id="{{item.id}}"
        mark:id="{{item.id}}"
    >
    	<text>点我</text>
    </view>
</view>
```

```typescript
Page({
    handleClick(e) {
        const id = e.mark.id
        console.log(id)
    }
})
```

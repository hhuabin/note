# ESM的import引入

1. 下载`https://res.wx.qq.com/open/js/jweixin-1.6.0.js`到`assets`下：`/assets/js/`中

2. `jweixin-1.6.0.js`源码末尾增加导出：

   ```typescript
   /* eslint-disable */
   // 把 this 改成 window
   export default window.wx || window.jWeixin;
   ```

3. `vite.config.ts`配置

   ```typescript
   resolve: {
       alias: {
           '@': new URL('./src', import.meta.url).pathname,
           'jweixin': new URL('./src/assets/js/jweixin-1.6.0.js', import.meta.url).pathname,
       },
   },
   ```

4. 增加`ts`类型声明

   ```typescript
   // jweixin.d.ts
   /* eslint-disable @typescript-eslint/no-explicit-any */
   
   declare module 'jweixin' {
       const wx: any
       export default wx
   }
   ```

5. 引入

   `ES Module (ESM)`支持`export default`

   ```typescript
   import wx from 'jweixin'
   ```

   动态引入

   ```typescript
   const { default: wx } = await import('jweixin')
   if (wx) {
       wx.miniProgram.navigateTo({
           url: `/pages/webPage/webPage?src=${advertisementItem.url}`
       })
   } else {
       Toast.show('jweixin is null')
   }
   ```

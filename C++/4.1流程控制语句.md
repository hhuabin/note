# 跳转

1. `break`表示跳出当前的流程控制语句，只能作用于`switch`或循环语句`while`、`do while`、`for`中，当代码遇到`break`时，会直接中断最近的`switch`或循环

   ```c++
   for (int i = 1; i < 10; i++) {
       if (i == 7) {
           break;       // 到7结束循环
       }
   }
   ```

2. `continue`表示继续执行当前循环，也就是中断循环中本次迭代，并开始下一次迭代。`continue`只能用在循环语句中

   ```c++
   for (int i = 1; i < 10; i++) {
       if (i == 7) {
           continue;     // 跳过7
       }
   }
   ```

3. `goto 标签;`表示无条件跳转到程序中另一条语句；不推荐使用，随意跳转比较危险，也容易造成死循环

   ```c++
   begin: int a = 0;           // 标签名可以取名 begin，也可以取名 hello，自定义即可
   ```

   ```c++
   int main() {
   	int x = 0;
   
   begin:
       do {
           cout << ++x << endl;
       } while (x < 3);              // while 只能打印到 3
   
   	if (x < 5) {                  // 可以打印到 5
   		goto begin;
   	}
   }
   ```

4. `return`中止函数运行并返回结果

   ```c++
   for (int i = 1; i < 10; i++) {
       if (i == 7) {
           return i;     // 返回 7
       }
   }
   ```






# 爱心图形

```c++
#include<iostream>
using namespace std;

int main() {

	// 爱心曲线方程 (x^2 + y^2 - a)^3 - x^2 + y^3 = 0
	double a = 0.8;
	// 定义绘图边界
	double bound = 1.3 * sqrt(a);

	double step = 0.05;
	for (double y = bound; y >= -bound; y -= step)
	{
		for (double x = -bound; x <= bound; x += 0.5 * step)
		{
			double result = pow((pow(x, 2) + pow(y, 2) - a), 3) - pow(x, 2) * pow(y, 3);
			if (result <= 0) cout << "*";
			else cout << " ";
		}
		cout << endl;
	}
	return 0;
}
```

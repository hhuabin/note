# 函数

```c++
#include<iostream>
using namespace std;

// 形参
int square(int x) {
	return x * x;
}

int main() {

    // 实参
	int res = square(3);

	cout << res << endl;

	return 0;
}
```



## `static`局部静态对象

函数内部变量只会被创建一次，**作用域只在函数内部**，但是生命周期贯穿整个程序执行（说人话就是：**保留数据**）

```c++
void callCount() {
	static int i = 0;

	cout << "函数被调用了" << ++i << "次" << endl;
}

int main() {

	for (int i = 0; i < 5; i++) {
		callCount();
	}

	return 0;
}
```



## 声明函数

有函数声明之后，函数可以放到其他文件中

```c++
#include<iostream>
using namespace std;

int square(int x);      // 前置函数声明
// int square(int);        // 也可以省略形参

int main() {

    // 实参
	int res = square(3);

	cout << res << endl;

	return 0;
}

// 形参
int square(int x) {
	return x * x;
}
```



## `.h`头文件

1. 我们可以把一些公共函数封装到一个文件中，然后用`.h`文件集中引入

   ```cpp
   // utils.h
   
   int square(int x);
   
   int clamp(int num, int max, int min);

2. 在代码文件中，把头文件引入即可

   ```c++
   #include "utils.h"
   
   int main() {
       int a = square(3)
   }
   ```

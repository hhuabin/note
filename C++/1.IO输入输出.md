# 输入输出

```c++
#include<iostream>
using namespace std;

int main() {
	std::cout << "Hello World" << std::endl;

	string name;

    // 使用输入操作符读取字符串
	std::cin >> name;
    // std::cin >> name >> name2;       // 以空格分隔

	std::cout << name << std::endl;
    
    return 0;
}

```





# 输入

1. 以空格分隔，获取单个字符串

   ```c++
   string str;
   std::cin >> str;
   std::cout << str << std::endl;             // 输出空格之前的字符串
   ```

2. 以空格分隔，输入多个字符串

   ```c++
   string str1, str2;
   std::cin >> str1 >> str2;
   std::cout << str1 << str2 << std::endl;     // 以空格分隔，输出两个字符串
   ```

3. `getline()`获取整行字符串

   ```c++
   #include<iostream>
   #include <string>
   using namespace std;
   
   int main() {
   	string str;
   	getline(cin, str);          // 以回车符为分隔，获取整行输入
   	cout << str << endl;
   }
   ```

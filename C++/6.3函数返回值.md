# 函数返回值

1. 返回**值的引用**（注意不要返回局部变量的引用）

   ```c++
   // 返回更长的字符串
   const string& longStr(const string& str1, const string& str2) {
   	return str1.size() > str2.size() ? str1 : str2;
   }
   
   int main() {
   
   	const string str = longStr("ABC", "DEF");
   
   	cout << str << endl;
   
   	return 0;
   }
   ```

2. 返回数组指针

   ```c++
   int (* funcName(int x))[5];            // 返回数组长度为 5 的数组指针
   ```

   简化函数声明

   ```c++
   typedef int arrayT[5];
   
   arrayT* funcName(int x);
   ```

   尾置返回类型

   ```c++
   auto funcName(int x) -> int(*)[5];
   ```

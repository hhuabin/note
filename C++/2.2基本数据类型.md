# 整形

## 1.`int`整形（4字节）

```c++
int a1 = 1;
std::cout << "int占 " << sizeof a1 << " 字节" << std::endl;
```

无符号整型，无符号少一个符号位，故能表示的 **正数** 范围更大以下

```c++
unsigned int a = 0;
```



## 2.`short`短整形（2字节）

```c++
short a = 1;
std::cout << "short占 " << sizeof a << " 字节" << std::endl;
```



## 3.`bool`布尔类型（1字节）

```c++
bool b = true;
std::cout << "bool占 " << sizeof b << " 字节" << std::endl;
```



## 4.`char`字符（1字节）

```c++
char str = '0';
std::cout << "char占 " << sizeof str << " 字节" << std::endl;
```



## 5.`long`长整型（4字节）

```c++
long a2 = 1;
std::cout << "long占 " << sizeof a2 << " 字节" << std::endl;
```



## 6.`long long`长整型（8-16字节）

```c++
long long a3 = 1;
std::cout << "long long占 " << sizeof a3 << " 字节" << std::endl;
```

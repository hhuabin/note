# 引用

可以解决指针使用麻烦的问题



## `int&`

`int&`引用也就是起个别名，不会占据新的存储空间。而且一旦绑定，就不可更改

```c++
int a = 10;

// 一旦绑定，就不可更改
int& ref = a;              // ref 即 a 的别名，操作 ref 即操作 a，这就是引用

cout << ref << endl;       // 10

ref = 20;

cout << a << endl;         // 20
```



## `const int&`常量的引用

```c++
const int a = 10;

const int& ref = a; 
```

1. 可以用字面值常量（10）做初始化

   ```c++
   const int& ref = 10; 
   ```

2. 可以用一个变量做初始化

   ```c++
   int a = 10;
   
   const int& ref = a;          // 该情况，不能使用ref修改a的值
   // ref = 20                  // 非法
   ```

3. 放宽数据类型

   ```c++
   double d = 2.1;
   
   const int& ref = d;          // ref = 2
   ```

   



## 引用 与 指针常量`int* const`

两者几乎一样，引用是指针常量的 语法糖，区别在于存储空间，引用不会占据存储空间

```c++
int a = 10;

const int& ref = a;       // 只是引用，并不会为引用（ref）开辟存储空间
int* const p = &a;        // 指针常量 p 将会占用一个指针存储空间
```

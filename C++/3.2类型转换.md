# 1. 隐式类型转换

1. 运算转换

   ```c++
   15.2 + 10;
   // 自动转换成 `double`
   ```

2. 初始化转换

   ```c++
   short a = 15.2 + 10;        // a = 30
   // `double`转`short`
   ```

3. 关系运算中类型转换

   反面教材，不要这么写哈

   ```c++
   string res = (0 < a < 100) ? "true" : "false";
   ```

   `0 < a`是真为`1`，`0 < a`是假为`0`，所以该运算`1 < 100`永远为真，永远返回`"true"`



# 2.强制类型转换

1. `C语言`风格；`(类型名称)值`

   ```c++
   int total = 10, num = 3;
   std::cout << (double)total / num << std::endl;      // 最后是 double 格式
   ```

2. `C++`函数强调风格；`类型名称(值)`推荐使用这种

   ```c++
   int total = 10, num = 3;
   std::cout << double(total) / num << std::endl;      // 最后是 double 格式
   ```

3. `C++`强制类型转换运算符；`static_cast<类型名称>(值)`

   ```c++
   int total = 10, num = 3;
   std::cout << static_cast<double>(total) / num << std::endl;      // 最后是 double 格式
   ```

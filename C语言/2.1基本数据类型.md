# 基本数据类型

| 类型                        | 说明                                                         |
| --------------------------- | ------------------------------------------------------------ |
| `char`                      | 字符类型，本质上也是 1 字节整数                              |
| `signed char`               | 有符号字符                                                   |
| `unsigned char`             | 无符号字符                                                   |
| `short` / `short int`       | 短整型                                                       |
| `unsigned short`            | 无符号短整型                                                 |
| `int`                       | 整型，取值范围 [$-2^{31}$, $2^{31}-1$]                       |
| `unsigned int`              | 无符号整型，取值范围 [0, $2^{32}-1$]                         |
| `long` / `long int`         | 长整型，平台相关类型（根据平台不同可能是32位(Windows)）<br /> **long long 永远 ≥ 8 字节** |
| `unsigned long`             | 无符号长整型，                                               |
| `long long`（C99）          | 更长的整型，取值范围 [$-2^{63}$, $2^{63}-1$]                 |
| `unsigned long long`（C99） | 无符号 long long，取值范围 [0, $2^{64}-1$]                   |

- `unsigned ` 只能表示 **0 和正数**，不能存负数。如果表示了负数，则会按照负数的**补码**取正值。如

  ```c
  unsigned int u = -1;
  
  printf("%u\n", u);         // 4294967295
  // -1 的补码是0xFFFFFFFF，转成 unsigned -> 4294967295
  ```

- `unsigned` 用符号位来表示数值

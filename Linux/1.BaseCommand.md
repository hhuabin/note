# 常用命令

### `shutdown`

```shell
shutdown         # 关机，会等一会再关机
shutdown -c      # 取消关机
shutdown -r      # 重启

ifconfig         # 查看计算机当前的网卡配置信息
ping ip地址      # 检测目标主机是否工作正常
```



### `ls` 查看

```shell
ls -a            # 列出目录所有文件，包含以.开始的隐藏文件
ls -t            # 以文件修改时间排序
ls -l            # 除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来
ls -h            # 配合 -l 使用，显示文件大小
```

```shell
ls -lhrt
```



### `tree` 树状结构显示

```shell
tree           # 以树状结构显示目录和文件
tree -L 1      # 显示层级深度，如这里显示层级1
tree -a        # 显示隐藏文件
tree -d        # 只显示目录

tree /a/b/c    # 查看指定目录
```



### `cd` 进入

```shell
cd            # 切换到当前用户的主目录
cd ~          # 切换到当前用户的主目录
cd .          # 保持当前目录不变
cd ..         # 切换到上级目录
cd -          # 在近两次目录中来回切换
```



### `pwd` 查看目录

```shell
pwd           # 查看当前所在目录
```



### `rm` 删除

```shell
rm -r           # 将删除某个目录以及其中所有的文件和子目录
rm -f           # 强制删除，忽略提示
rm -i           # 进行交互式删除，提示确认删除

rmdir directory           # 删除 directory 目录，前提是空的
rmdir -p a/b/c            # 如果 a/b/c 都为空，则删除
```



### `touch` 新建文件

如果文件不存在，创建空白文件；如果文件已存在，修改末次修改日期

```shell
touch test.txt
touch a.txt b.txt c.txt   # 创建多个文件
```



### `mkdir` 新建文件夹

```shell
mkdir directory      # 创建 directory/ 文件夹
mkdir -p a/b/c       # 创建多级文件夹
```



### `cp` 复制文件

```shell
cp a.txt b.txt           # 把 a.txt 复制成 b.txt，如果 b.txt 已存在，则会被覆盖
cp a.txt b.txt demo/     # 复制多个文件到目录

cp -i           # 如果文件将覆盖目标中的文件，他会提示确认
cp -r           # 这个选项会复制整个目录树、子目录以及其他
cp -v           # 显示文件的复制进度
cp -a           # 保留文件属性，如权限、时间等
```



### `mv` 移动

```shell
mv a.txt demo/       # 移动文件
mv a.txt b.txt       # 重命名文件
mv old_dir new_dir   # 重命名目录

mv -i                # 覆盖前询问
mv -a                # 显示移动过程
mv -f                # 强制移动，不询问
mv -n                # 不覆盖已存在的文件
mv -v                # 显示文件的移动进度

mv -n file1.txt /home/user/documents/
```



### `cat` 查看文件内容

```shell
cat text.txt               # 查看文件内容
cat -n test.txt            # 对输出的行进行编号
cat -b test.txt            # 只给非空行编号

cat a.txt b.txt > c.txt    # 拼接文件；把 a+b 合并到 c
cat app.log                # 查看日志
```



### `grep` 文本搜索工具

```shell
grep hello test.txt       # 在 test.txt 中搜索 hello

grep -i          # 忽略大小写
grep -n          # 显示行号
grep -r          # 递归搜索目录（超常用）
grep -v          # 显示不匹配内容
grep "^root" /etc/passwd        # 正则搜索
```



## 系统信息

   ```shell
   date

   ```



## 磁盘信息

```shell
df              #

```



## 进程信息

```shell
ps aux               # 查看进程详细状况
```

```shell
kill 进程号           # 终止进程
kill -q 进程号        # 强制终止进程
```

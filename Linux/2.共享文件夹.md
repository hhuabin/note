# 开启共享文件夹

1. 虚拟机中需要配置开启共享文件夹

   ```
   设置 -> 选项 -> 共享文件夹
   ```

   



## CentOS

1. 安装 `open-vm-tools` 和 `fuse` 工具

   ```bash
   sudo dnf install -y open-vm-tools open-vm-tools-desktop fuse3
   ```

   说明：

   - `open-vm-tools`：VMware 官方开源工具
   - `open-vm-tools-desktop`：桌面环境支持（建议装）
   - `fuse3`：新版本系统通常是 fuse3，不一定叫 fuse

2. 启动 `Open VMware Tools` 服务并设置服务**开机自启**

   ```bash
   sudo systemctl enable --now vmtoolsd
   ```

   ```shell
   systemctl status vmtoolsd      # 检查状态
   ```

3. 使用 `vmhgfs-fuse` 挂载共享文件夹（临时挂载）

   ```bash
   sudo vmhgfs-fuse .host:/ /mnt/hgfs
   ```

   ```shell
   sudo vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other       # 允许虚拟机上的非 root 用户也能访问共享文件夹
   ```

5. 验证共享文件夹

   ```bash
   ls /mnt/hgfs/
   ```

5. 开机自动挂载（确保 `vmtoolsd` 是开机自启状态）

   ```shell
   sudo vim /etc/fstab
   ```

   在文件中追加

   ```shell
   .host:/ /mnt/hgfs fuse.vmhgfs-fuse allow_other,defaults,_netdev 0 0
   ```

   验证

   ```shell
   sudo mount -a
   ```

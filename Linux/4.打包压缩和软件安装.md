# 打包压缩

## 打包

1. 打包

   ```shell
   tar -cvf project.tar project/          # 把 project 目录打包到 project.tar 文件
   ```

   - c：打包
   - x：解压/提取
   - v：显示过程
   - f：指定文件名
   - z：`gzip` 压缩
   - C：指定目录

2. `gzip` 压缩

   ```shell
   tar -zcvf project.tar.gz project/
   ```

除 `gzip` 压缩之外还有很多其他的压缩方式，如 `bzip2`，`zip` 等，需要再查即可



## 解包

1. 解包

   ```shell
   tar -xvf project.tar
   ```

2. `gzip` 解压

   ```shell
   tar -zxvf project.tar.gz
   ```

3. 指定解压目录

   ```shell
   # tar -zxvf project.tar.gz -C 目录
   tar -zxvf nginx.tar.gz -C /usr/local/
   ```





# 软件安装

在 `CentOS ` 中常用 `dnf | yum(旧版)` 安装软件

`Ubuntu ` 则使用 `apt`

## 安装软件

1. 安装

   ```shell
   # sudo dnf install 软件名
   sudo dnf install nginx
   
   sudo apt install nginx    # Ubuntu
   ```

2. 删除软件

   ```shell
   sudo dnf remove nginx
   
   sudo apt remove nginx     # Ubuntu
   ```

3. 搜索软件

   ```shell
   dnf list installed
   ```

4. 更新软件

   ```shell
   sudo dnf update
   
   sudo apt update           # Ubuntu
   sudo apt upgrade          # 升级软件
   ```
